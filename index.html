Projects

SPCA Cat Webscrapper

This project was made to keep track of the cats being put up for adoption on the SPCA website. The program is written in Python and uses the inbuilt library of SQLite3 for the data storage. The cat's details are scrapped from the websites HTML which are then cross-checked with the data in the database to determine whether the cat is new, any cats in the database that arent in the websites data will be deleted. The program is run 4 times thoughout the day and any update is formulated into an HTML formatted email and is sent to me where I can be notified of the change.
